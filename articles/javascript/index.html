What is JavaScript?

JavaScript is a very powerful client-side scripting language. JavaScript is used mainly for enhancing the interaction of a user with the webpage. In other words, you can make your webpage more lively and interactive, with the help of JavaScript. JavaScript is also being used widely in game development and Mobile application development.

Javascript is used to increase user interaction to the website. It is used to change or present content based on user inputs. Some basic examples of such interaction include but not limited to:

  • Showing or hiding content on demand
  • Menu
  • Drag and drop features
  • Form validation
  • fetching data based on input from users
  • animations

how to start with javascript

Javascript is written inside <script> tag. Anything written inside the tag is executed. Script tag is inserted as <script type="text/javascript"></script>
A script tag can be inserted inside <head> or just before closing the <body> tag

The difference between the two is,:-

  • the script in the <head> runs before the DOM structure is constructed and the later is run after the DOM is constructed
  • the script in <head> is render blocking i.e. stops the page from loading and should be very light and minimal.

Dialogue boxes

To check there are basic popup boxes in-built in javascript that involves very basic user interaction. They involve little to no knowledge of web coding. They also require no installation. These dialogue boxes popup on the user screen and expecting an input. Call to these are synchronous which mean, the program does not execute further until the input is received.

Alert

Alert boxes, as the name suggest are used to send an alert signal to the users. The box contains a text message. User is required to click ok to dismiss. 

alert(""); 
alert("This is an alert box");

Confirm

Confirm boxes are little more advanced than alerts. Like alerts, confirm boxes contain a text message . user now has an option to ok or cancel the message. Additionally, confirm also returns true or false accordingly for further calculation

var x = confirm(); 
var x = confirm("Do you confirm??");

Prompt

Prompt boxes are advanced. They take text input from the user and return the value. 

var x =  prompt();

You can add a customised message before receiving the input.

var x =  prompt("Please enter your name");

There are many options to interrupt the user with important messages about the application flow. There are native alert dialogs you can prompt using JavaScript or craft your own using CSS and HTML. Just make sure you don’t abuse the power to display messages to the user.
