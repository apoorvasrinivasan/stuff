Simple Todo in plain Javascript

TO-DO list

Todo list are the first step to coding after hello world. TO-do lists are basically lists that are created by users mentioning the tasks they need to get done. They can assign priority or category to them. As they keep doing the tasks, user have an option of marking it as done so as to show their progress.

In this tutorial we are going to make a very simple todo list using basic vanilla JS. To start, create and save an html file. In order to do that, you can refer my previous tutorial here

HTML

To begin with , let’s write the basic html code for entering the list.

<input type='text' class='todo' placeholder = 'enter an item'>

To add text before it, you can add a label

<label> Enter an item <input type='text' class='todo'> <label>

Next, we are going to display the do to list. For that, we create a ul element and give it a nice id. Initially, it will be blank list and will not effect the way your document looks like.

    JAVASCRIPT

    First functionality to add is to enter the input to the displayed list. We need the javascript function to

    • add a list item to the ul
      • access the ul
      • create a li element
      • set its status as tbd
    • the list item should contain the text entered by the user
      • set the value of li as the input box
      • clear the input box
    • this should happen as soon as the user finishes typing.
      • trigger the function after the input box value changes.

    Lets write the function

    var addTodo = function (el) {
  // get the value of the input
  var todoValue = el.value;
  el.value = '';

  // get the UL element
  var ul = document.getElementById("TodoList");

  // create a list item
  var li = document.createElement("li");
    li.setAttribute('data-status','tbd');
  li.innerHTML = todoValue;

  // append to the ul
  ul.appendChild(li);
};

    Now we need to set to execute the function only when the user finishes typing. for that, we add the event handler, onchange.

    <input type='text' class='todo' onchange='addTodo(this)'/>

    Next step is to add checkboxes to the list items so that user can mark any item as done.

      // add the checkbox
  check = document.createElement("check");
  check.type='checkbox';
  li.appendChild(check);

    To change the status of the task

    To be able to change the status of the task, the function needs to

    • mark the task as done
      • access the checkbox that is clicked
      • access the list item
      • change the data attribute from tbd to done
      • add necessary CSS to visually represent the same.
    • trigger when the checkbox is checked
      • tell javascript to listen to any event that changes the condition of the checkbox.
    • undo the changes when the checkbox is unchecked.

    For this, first we add a event listener to the checkbox we just created as

    input.addEventListener('change',changeStatus);

    Change is the event action, and changeStatus is the function to be executed.

    Lets write this new function that changes the status of the li

    var changeStatus = function(){

   // get the list item that triggered this event.
   el = event.target;
   var checked = el.checked;

  // get the parent li element
  var li = el.parentElement;
  // if the checkbox is checked
   if(checked)
     li.setAttribute('data-status','done');
  else
    li.setAttribute('data-status','tbd');

 }

 // CSS
 #TodoList li[data-status="done"] {
    color: grey;
    font-style: italic;
    text-decoration: line-through;
}
